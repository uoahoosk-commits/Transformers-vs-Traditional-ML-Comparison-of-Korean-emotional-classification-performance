# 트랜스포머 vs 전통 ML: 한국어 감성분류 성능 비교

사전학습된 트랜스포머 모델과 전통적 기계학습 모델(TF-IDF + 로지스틱 회귀)을 동일한 한국어 감성분류 과제에서 비교한 프로젝트입니다.

## 개요

영화 리뷰가 긍정인지 부정인지 분류하는 과제를 두 가지 방식으로 풀고 정확도를 비교했습니다. 핵심 질문은 다음과 같습니다.

> **사전학습된 트랜스포머는 밑바닥부터 학습한 전통 ML 모델보다 얼마나, 왜 더 정확한가?**

## 데이터셋

- **NSMC** (Naver Sentiment Movie Corpus): 네이버 영화 리뷰 + 긍정(1)/부정(0) 라벨
- 전체 약 20만 건 중 **학습 3,000건 / 평가 1,000건**을 무작위 추출 (`random_state=42`)

## 비교 모델

|모델              |방식                     |학습            |
|----------------|-----------------------|--------------|
|TF-IDF + 로지스틱 회귀|글자 단위 n-gram 특징 + 선형 분류|밑바닥부터 학습      |
|사전학습 트랜스포머      |NSMC 미세조정 BERT 계열      |예측만 (추가 학습 없음)|

## 결과

|모델              |정확도      |F1 점수    |
|----------------|---------|---------|
|TF-IDF + 로지스틱 회귀|0.801    |0.789    |
|**사전학습 트랜스포머**  |**0.884**|**0.883**|

트랜스포머가 정확도 약 **8.3%p**, F1 약 **9.4%p** 앞섰습니다.

## 주요 발견

- 전이학습(사전학습)을 거친 트랜스포머가 적은 라벨만으로도 미세한 표현 차이를 잘 구분했다.
- 어텐션 메커니즘 덕에 부정어·문맥을 더 정확히 포착했고, 이는 F1 격차가 정확도 격차보다 컸던 점에서 드러난다.
- 전통 ML도 0.80을 기록 — 직설적 표현이 많은 과제에서는 단순한 모델도 충분히 실용적일 수 있다.

## 한계

- 평가 표본이 1,000건으로 작고, 영화 리뷰라는 단일 도메인에 한정됨
- 두 모델의 학습 조건이 비대칭적(사전학습 vs 밑바닥 학습) — “구조 자체의 공정한 비교”가 아니라 “실제 사용 방식대로의 성능 비교”로 이해해야 함
